# Introducing FormsIO

We built FormsIO because every existing form builder feels like filling out a spreadsheet.

Respondents drop off. Completion rates tank. You lose data.

FormsIO flips this — every form renders one question at a time, with smooth transitions and a conversational flow that keeps people engaged.

## What ships in v1

- **18+ question types** — short text, email, rating, scale, file upload, address, currency, and more
- **Admin CRUD** — create, edit, reorder sections and questions from a clean dashboard
- **Public slug URLs** — share `/f/your-form-slug` instantly
- **Hybrid display modes** — one-at-a-time (Typeform-style) or full-section scroll

## Coming soon

- Conditional branching (Phase 3) — show/hide questions and jump sections based on answers
- Webhooks (Phase 4) — push every response to any API with custom key mapping
- File uploads (Phase 4) — S3-backed uploads with presigned URLs per question

We're building in public. Follow along.
